# Reddit Personal Research App

This is a personal, non-commercial, read-only exploratory research project.

Purpose:
To understand aggregate public discussion trends and broad consumer needs across selected public Reddit communities.

The app will:
- use approved Reddit API access
- authenticate with OAuth
- use a unique descriptive User-Agent
- retrieve only limited public posts and comments
- generate private aggregate research notes

The app will not:
- post, comment, vote, or message users
- scrape Reddit HTML pages
- bypass Reddit API limits or access controls
- profile, target, or re-identify Redditors
- infer sensitive personal characteristics
- redistribute, sell, license, or commercialize Reddit data
- use Reddit data for advertising, lead generation, or outreach
- train, fine-tune, evaluate, or sell AI/ML models

Data handling:
Only the minimum public data necessary for personal aggregate analysis will be used. Deleted or removed Reddit content will be removed from local records when identified.
