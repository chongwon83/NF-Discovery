# NF-Discovery

This is a personal, non-commercial, read-only Reddit Data API research project.

Purpose:
To understand aggregate public discussion trends and broad consumer needs across selected public Reddit communities.

This project does not:
- post, comment, vote, or message users
- profile or target individual Redditors
- redistribute raw Reddit data
- train or fine-tune AI/ML models
- bypass Reddit rate limits or access controls

Data handling:
Only minimal public fields are used for aggregate personal analysis. Deleted or removed Reddit content will be removed from local records.
